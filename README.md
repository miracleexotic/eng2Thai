# eng2Thai Library

  - convert eng to thai
  - code : [input eng encoding]
  - covert : [output thai decoded]

### Install

Use pip

```sh
pip install eng2Thai
```

### Used
  - import eng2Thai
  - 2 method = [decode(string), run()] 
  - use (-1) to exit
```sh
>>>import eng2Thai
>>>eng2Thai.run()
code : l;ylfu8iy[
สวัสดีครับ
code : -1
>>>eng2Thai.decode("l;ylfu8iy[")
'สวัสดีครับ'
```
